Strem widgets, only local!

 

Introduction

This is a mini-implementation of a web server combined with a simple python script that is designed to help local fragforce orgs get some neat stream widgets without having to rely on an external service.

The system relies on a python script to pull donations from the team api and the user api records, and keeps them locally, while the pages use JS to read those files and display the data accordingly. As this is a general webserver, additional non-data related pages can be created and used as widgets as well.
